[ WHAT IS CUEFLEX? ] [ GET MOBILE APP / APK ]

CueFlex is a serverless, zero-install peer-to-peer file transfer utility for rapid data movement between desktop terminals and mobile devices.

• ZERO CLOUD STORAGE: Files stream directly between devices via WebSocket memory relay without server caching.
• E2E ENCRYPTED: 256-bit AES-GCM encryption derived from the ephemeral session ID ensures zero interception.
• INSTANT PAIRING: Point your CueFlex app or iPhone camera at the QR code to connect in <1 second.
[ PAIRING PROCEDURE ]
01 Open CueFlex mobile app or iPhone camera
02 Scan the QR code displayed on the right
03 Connected — drop files or browse target directory across full screen
